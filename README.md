# ChatRealTime 💬

> Chat em tempo real com WebSockets, salas privadas e confirmacao de entrega.

## Sobre

Aplicacao de chat utilizando Socket.IO para comunicacao bidirecional em tempo real. 
Suporta salas privadas, mensagens com confirmacao de entrega, upload de arquivos 
e indicador de digitacao.

## Stack

Socket.IO • Node.js • MongoDB • Express • Sharp

## Funcionalidades
- Mensagens em tempo real com confirmacao (✓✓)
- Salas privadas com convite
- Upload de arquivos com compressao (Sharp)
- Indicador de digitacao
- Historico persistente (MongoDB)
- Suporte a emojis e midia

## Metrica
Latencia de mensagem inferior a 50ms em rede local.
